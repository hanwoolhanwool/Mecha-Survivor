# 무기 목록 및 상세 (2026-07-18 기준)

> **수치 출처**: `Assets/_Project/ScriptableObjects/Weapons/*.asset` (실측 덤프).
> 밸런싱 수정은 코드가 아니라 **에셋 인스펙터**에서 한다 — 재빌드 없이 반영된다.
> 무기 구현체는 `Assets/_Project/Scripts/Gameplay/Weapons/`.

## 역할(Role) 체계 — 로테이션의 자리

| 역할 | 의미 | 해당 무기 |
|---|---|---|
| **Sustain** (주력) | 로테이션의 바닥. 홀드 연사 허용 | 개틀링, 레이저 캐논 |
| **Burst** (화력) | 무리 정리 | 미사일 포드, 산탄 캐논, 클러스터 폭탄 |
| **Finisher** (결정타) | 엘리트/보스 삭제 | 레일건, 대출력 빔, 궤도 폭격 |
| **Control** (유틸) | 상황을 만든다 | 그래비티 웰, EMP 필드 |

발사 자원은 없다 — **쿨다운이 전부**다. 실제 쿨다운은 나눗셈 모델
`기본 ÷ (1 + 쿨감 합계) ÷ 지상 배율` 로 계산된다 (`CooldownMath.cs`).

## 무기별 상세

### 개틀링 (Sustain)
홀드 연사(0.12초 간격), 탄 퍼짐 3°, 탄속 80. 빈 시간을 메우는 로테이션의 바닥.
강화 시 데미지/연사 상승, Lv5에서 가속 회전. 조합 「가열식 개틀링」의 재료.

### 레이저 캐논 (Sustain)
즉시 명중(히트스캔) 펄스 레이저. 발사 시차 0.06초의 연발이며, 강화할수록 부채꼴로 갈라져
Lv5에는 5갈래가 된다.

### 미사일 포드 (Burst)
발사 스타일 `VerticalPopup` — 수직 팝업 → 정렬 → 곡선 유도(유도 강도 200°/초).
발사 시차 0.08초로 "두두두둥" 연출. 4발에서 시작해 Lv5에 16발. 조합 「다중 락온」의 재료.

### 산탄 캐논 (Burst)
근거리(사거리 45) 원뿔 산탄, 퍼짐 16°. 발사 반동으로 기체가 밀리고(`MechaController.AddImpulse`)
펠릿이 적을 밀쳐낸다(넉백). Lv5: 화염방사 웨이브.

### 클러스터 폭탄 (Burst)
공중 분열 융단폭격 — 모탄 5발이 자탄으로 분열한다. Lv5: 자탄이 또 터진다(2차 분열).

### 레일건 (Finisher)
차징 0.6초 후 관통 저격. 발사 순간 카메라 킥(`WeaponDischargedEvent`).
Lv5: 명중 지점 2차 폭발.

### 대출력 빔 (Finisher)
차징 1초 → 1.8초간 지속되는 관통 빔. 데미지 70은 **초당** 데미지이며 초당 10회 판정.
일직선상의 적을 한 줄로 증발시킨다. 강화 시 빔 굵기가 크게 성장(+50%/레벨).

### 궤도 폭격 (Finisher)
지상 마커 표시 후 빛기둥 낙하. 착탄 시 화면 플래시 + 버섯구름(`HeavyImpactEvent`).
Lv5: 융단 폭격 5연발.

### 그래비티 웰 (Control)
적을 한 곳으로 빨아들이는 중력장 투사체(탄속 28). 미사일·빔 콤보의 핵심 셋업기.

### EMP 필드 (Control)
광역 슬로우 + 원거리 적 사격 봉쇄 필드. Lv5: 체인 감전.

## 무기 슬롯

- 시작 **2슬롯**, 「무기 슬롯 확장」 파츠(최대 2회)로 **최대 4슬롯**.
- 발사 키: 슬롯1 = 좌클릭, 슬롯2 = 우클릭, 슬롯3 = Q, 슬롯4 = E.
- 빈 슬롯이 없으면 새 무기 파츠는 3택에 등장하지 않는다 (`PartUpgradeData.CanOffer`).
