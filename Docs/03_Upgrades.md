# 레벨업 강화(3택) 목록 (2026-07-18 기준)

> **수치 출처**: `Assets/_Project/ScriptableObjects/Upgrades/*.asset`, `Recipes/*.asset` 실측 덤프.
> 로직: `UpgradeService`(후보 추출) → `UpgradeRoller`(가중치 비복원 추첨) → `LevelUpPanel`(UI).

## 동작 규칙

- 레벨업 시 게임이 멈추고 **3택**이 열린다. 후보 = 미보유 획득안 + 보유 강화안(레벨 미달) + 조건 충족 조합안.
- **가중치(Rarity)** 가 클수록 자주 등장. 같은 항목 재선택 = 강화 (기본 최대 Lv5).
- **리롤: 레벨업당 1회** (「다시 뽑기」 버튼, `RerollBudget`).
- 제안 필터: 빈 무기 슬롯이 없으면 신규 무기 미등장, 슬롯이 최대(4)면 슬롯 확장 미등장.
- 쿨감은 나눗셈 모델 — `실쿨 = 기본 ÷ (1 + 쿨감 합계) ÷ 지상 배율`. 아무리 쌓아도 0이 되지 않는다.

## 조합 3종 (Recipe — 재료를 소비하고 상위 항목 획득)

재료 레벨 조건을 채우면 3택에 **가중치 ×3 보너스**로 등장한다 (`_combinationWeightBonus`).
선택 시 재료가 소비된다 (`UpgradeInventory.Combine`).

| **다중 락온 시스템** | 미사일 포드 Lv3 + 고속 미사일 장전기 Lv2 | 미사일 전용 쿨감 **+80%** |

## 경험치 곡선 (레벨업 페이스)

- Lv N → N+1 요구치 = `5 + 4×(N−1)` (`ExperienceCurve`, 인스펙터에서 base/growth 조정 가능).
- 경험치 소스: 잡병 1 / 드론 2 / 포탑 4 / 엘리트 25 / 보스 100 (젬 드랍).
